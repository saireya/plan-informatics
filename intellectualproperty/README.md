# 知的財産(6月下旬)
## 単元の目標
多くの情報が公開され流通している現状を認識させるとともに、情報を保護することの必要性とそのための法規及び個人の責任を理解させる。また、情報の収集や発信などの取扱いに当たっては個人の適切な判断が重要であることを理解させる。
